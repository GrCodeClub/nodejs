      Web Server

      var http = require('http');
http.createServer(function (req, res) {
  res.write('Hello World!'); 
  res.end();
}).listen(8080);

      Η συνάρτηση http.createServer(), θα εκτελεστεί όταν κάποιος προσπαθήσει να αποκτήσει πρόσβαση στον υπολογιστή στη θύρα 8080.

      Add an HTTP Header

      var http = require('http');
http.createServer(function (req, res) {
  res.writeHead(200, {'Content-Type': 'text/html'});
  res.write('Hello World!');
  res.end();
}).listen(8080);

      Το πρώτο όρισμα της μεθόδου res.writeHead() είναι ο κωδικός κατάστασης, το 200 σημαίνει ότι όλα είναι εντάξει, το δεύτερο όρισμα είναι ένα αντικείμενο που περιέχει τα response headers.

      Το αντικείμενο {'Content-Type': 'text/html'} δηλώνει τον τύπο περιεχομένου (Content-Type) της απάντησης. Στη συγκεκριμένη περίπτωση, ορίζει ότι το περιεχόμενο είναι τύπου κειμένου HTML. Αυτό σημαίνει ότι το περιεχόμενο που θα σταλεί προς τον πελάτη θα ερμηνευτεί ως HTML και θα εμφανιστεί αντίστοιχα στον περιηγητή.

      Διαβάστε τα Query String

      Η συνάρτηση http.createServer() έχει ένα όρισμα req που αντιπροσωπεύει το αίτημα από του client, ως object

      var http = require('http');
http.createServer(function (req, res) {
  res.writeHead(200, {'Content-Type': 'text/html'});
  res.write(req.url);
  res.end();
}).listen(8080);
